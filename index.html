Statistics Web Page

Combining Statistics and Design

This page collates statistics on refugee and asylum seeker populations in Australia and the world for the user to easily find, compare and engage with vital statistics

The page is published using WordPress, with HTML and CSS used for the page layout. The graphs are visualised on the web page through the Google Visualisation API that is made using JavaScript. The graphs are populated by sending an SQL query to a datasource, the datasource is this case are Google Spreadsheets.

The following are the main sources used:

  • UNHCR Global Trends 2013,
  • UNHCR Mid Year Asylum Trends 2014, and
  • DIBP Monthly Immigration Detention Statistics

This is an ongoing project, not just to keep the statistics up to date, but to also to see how the users are engaging with the data. What can change:
