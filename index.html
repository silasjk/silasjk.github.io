ASRC Statistics

The Asylum Seeker Resource Centre Statistics Page

Designing Statistcs for Webpages

This page collates statistics on refugee and asylum seeker populations in Australia and the world for the user to easily find, compare and engage with vital statistics

The page is published using WordPress, with HTML and CSS used for the page layout. The graphs are visualised on the web page through the Google Visualisation API that is made using JavaScript. The graphs are populated by sending an SQL query to a datasource, the datasource is this case are Google Spreadsheets.

The following are the main sources used:

  • UNHCR Global Trends 2013,
  • UNHCR Mid Year Asylum Trends 2014, and
  • DIBP Monthly Immigration Detention Statistics

This is an ongoing project, not just to keep the statistics up to date, but to also to see how the users are engaging with the data. What can change:

Web Content
Timeline of Events

Timeline of Refugee and Asylum Seeker Events

Using a timeline to present information

Since August 2012 there have been a number of events that have created significant changes in policy and attitudes towards asylum seekers and refugees in Australia. This timeline gives a brief summary of these events, sort them by date and provides each event with further content, such as pictures and videos.

The design of this timeline provides clean layout, and breaks up otherwise long periods of text with videos and pictures. Thios design is made to keep the readers attention for lnger periods.

The timeline was created using HTML and CSS and published through WordPress. To make it easier for the reader, the event boxes are all down the right hand side of the timeline, rather than on both the left and right hand side of the timeline and are in descending date order.

Further changes will be made to this timeline, such as continue to update with new events and categorising each event allowing the reader to view or sort only by that category.

See the Timeline of Events on the ASRC’s website.

Graphics
#StandUpToProtect

#StandUpToProtect

Using graphics in advocacy campaigning

The Asylum Seeker Resource Centre launched a campaign to stop the Migration and Maritime Powers Legislation Amendment (Resolving the Asylum Legacy Caseload) Bill 2014, that if passed would have huge negative effects on the lives of refugees and asylum seekers in Australia.

These graphics were created to be shared on social media, Facebook and Twitter, to highlight the negative effects on asylum seekers and refugees should the bill be passed.

The graphics were made using Adobe Illustrator and photos if needed were edited in Adobe Photoshop. Photos under the creative commons licence were used. In most cases a photo containing a face was used as faces are the best way to grab the attention of the reader. The copy was provided by the ASRC staff and were based on case studies. Photos are credited to the photographer.

Visit the ASRC's StandUpToProtect webpage to view the other graphics

ASRC Resources

Myths Facts and Solutions

Document presentation (currently working on)

One of the largest resources the Asylum Seeker Resource Centre has created is 'Myths Facts and Solutions', a publication that dispells the most common myths about asylum seekers and refugees. The document is updated each year, due to changes in statistics and changes in the common myths.

This project requires strong research skills, exceptional writing and editing skills, attention to detail and an advanced knowledge of Adobe InDesign. The final design of this document, which will be made in InDesign, will need to be engaging, clear and consistent to engage the reader.

On presentation on the website the document will need to be split up and presented as individual myths. This allows the reader to quickly find the myths they want to know more about. These pages will require HTML, CSS and some JavaScript and the Google Visualisation API.

Go here to see the current version of Myths, Facts and Solutions.
